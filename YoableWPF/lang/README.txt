語言文件說明
============

此文件夾包含 Yoable 的語言翻譯文件（INI 格式）。

文件格式：
---------
- 每個語言對應一個 .ini 文件
- 文件名格式：{語言代碼}.ini（例如：zh-TW.ini）
- 文件編碼：UTF-8

支持的語言：
----------
- en-US: 英文（硬編碼在程序中，無需 INI 文件）
- zh-TW: 繁體中文（zh-TW.ini）
- zh-CN: 簡體中文（zh-CN.ini）
- ja-JP: 日文（ja-JP.ini）
- ru-RU: 俄文（ru-RU.ini）

如何添加新語言：
--------------
1. 複製現有的 .ini 文件作為模板
2. 重命名為 {語言代碼}.ini
3. 修改 [LanguageInfo] 區段中的 Name 和 NativeName
4. 翻譯所有區段中的值
5. 重新啟動應用程序，新語言會自動出現在語言列表中

文件結構：
---------
[LanguageInfo]     - 語言基本信息
[Common]          - 通用按鈕和文字
[MainWindow]      - 主窗口
[Settings]        - 設置窗口
[UI]             - UI 相關
[Menu]           - 菜單項
[Main]           - 主窗口面板
[Status]         - 狀態欄
[Mapping]        - 模型類別映射對話框

注意事項：
---------
- 如果某個語言文件不存在，該語言不會出現在語言列表中
- 如果翻譯缺失，會自動回退到英文
- 修改 INI 文件後需要重新啟動應用程序才能生效
- 使用 \n 表示換行，\t 表示製表符
